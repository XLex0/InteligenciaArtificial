Funcionalidad: Mostrar tono de piel

calculoPiel-py es un programa que nos permite saber el tono de
piel de una persona. Se usa la libreria cv2 por lo que es necesario 
hacer un pip install de la librería, si no se la tiene instalada

¿Cómo usar?
Se debe ingresar la imagen a ser analizada en base a su ruta
EL fondo debe de ser de un color distinto a un tono de piel para
evitar errores
Se muestra una imagen de la foto y la seleccion analizada

Luego la mascara que son los colores qeu se detecta como piel

Y en base a la moda se elige el tono de piel que se devuelve
con un cuadrado indicando el color.
